## Predicting Next Year's Best Rated Board Games

### Overview

This project explores the factors associated with highly rated board games using data from BoardGameGeek (BGG). We apply Bayesian regression models to study how game complexity, mechanics, domains, and publication year influence player ratings.

Our goal is to identify the characteristics of successful games and investigate whether it is possible to predict the features of next year's best rated board games.

### Dataset

The analysis uses the Board Games Dataset from Kaggle, containing information on more than 20,000 board games, including ratings, complexity, mechanics, domains, and publication year.

## Methods

- Data preprocessing and feature engineering
- Bayesian regression with PyMC
- MCMC sampling using NUTS
- Posterior analysis and convergence diagnostics
